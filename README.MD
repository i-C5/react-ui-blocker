# React UI Blocker

A simple loader animation to block UI view while AJAX request under process.

### Installation

`npm i @i-c5/react-ui-blocker`

or

`yarn add @i-c5/react-ui-blocker`

### Usage

```
    import UiBlocker from @i-c5/react-ui-blocker

    <UiBlocker block="true">
        <>
        {children}
        </>
    </UiBlocker>
```
